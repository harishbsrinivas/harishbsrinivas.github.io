Fedora Core diskless install

I downloaded Fedora core 6 Last week and instead of burning the ISO images to CD'S I wanted to do a harddisk based install. A Disk less install can be achieved by the following method

1. Copy the ISO files to the root or c: drive of your Hard disk under a directory name zod.
2. Burn the First CD image to a CD using a CD burner,you can use deepburner or CDCC.
3. Boot from CD and at the boot prompt type Linux askmethod.
4. select hda1 and type in zod for the directory name.
5. Install linux.
