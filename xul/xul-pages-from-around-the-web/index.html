XUL pages from around the web

I wanted to see how other people were using XUL and did a search on Google for "ext:XUL"

I found close to 15,400+ results here are a few pages that I found interesting

  • An interactive map viewer : Pretty basic viewer.
  • A CSS programmers repertoire : A homepage created in XUL.
  • XUL based memory games : A basic set of games more of a learning project.
  • Google search interface : Google's search page in XUL.
  • A password generator : Generates a site specific password based on a master password.
  • Couchdb browser : Seems to be a very early version of a database administration tool.
  • Calender help : The XUL version of calender help for Sunbird.
  • Another game : Another game implemented in XUL
  • XUL presentation : A XUL based presentation on GNU tools.
  • Boooofy ??? : I don't know what the heck this is but its implemented in XUL.
  • An online registration form : An online registration form in XUL (German site).
  • Kiosk project : A kiosk project help file implemented in XUL.
  • One click : Another implementation of homepage in XUL.
  • RPNCalc : A RPN calculator in XUL.
  • Linear inequations grapher : A charting program for system of linear inequations.
  • Histogram tool : A Histogram visualization tool implemented in XUL.
  • Zope home page : Zope homepage implemented in XUL.
  • Ontology db : A clinical database front end in XUL.
  • Simple calculator : A simple calculator in XUL.
  • Server post : A implementation of the POST method in XUL.

The above examples are not much to look at but if you are learning XUL then they are a great help, in case you want to downlaod any of the XUL code just open the link in IE and you can download and examine the code.
